# CV-Project
 # 🎨 AirCanvas Pro - 3D Gesture Drawing Studio with Structure from Motion

## Project Overview

AirCanvas Pro is a revolutionary 3D drawing application that combines **Structure from Motion (SfM)** for 3D reconstruction with **real-time hand gesture recognition** for intuitive drawing in space.

## ✨ Key Features

- **🎯 Gesture-Based Drawing**: Draw in air using hand gestures captured by camera
- **🏗️ 3D Reconstruction**: Convert 2D images to 3D models using SfM pipeline
- **🖌️ Multiple Drawing Tools**: Brush, Fill, Smudge, Text, Move, and Effects
- **🤝 2-Way Interaction**: Camera captures gestures → Canvas responds in real-time
- **📦 3D Object Integration**: Import and manipulate 3D objects from SfM reconstruction
